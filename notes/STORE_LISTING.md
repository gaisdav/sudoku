# Store listing

## App name (display name)

**Sudoku Offline – Brain Puzzle**

- **Android:** set in `android/app/src/main/AndroidManifest.xml` (`android:label`) and/or `android/app/src/main/res/values/strings.xml`; can be overridden in Google Play Console.
- **iOS:** set in `ios/Runner/Info.plist` (`CFBundleDisplayName`); can be edited in App Store Connect.

---

## Keywords

Use these when filling store listing fields (Play Console descriptions, App Store Connect keywords):

- sudoku
- puzzle
- brain game
- logic
- numbers
- offline sudoku

**App Store Connect** — paste in the "Keywords" field (comma-separated, no spaces, up to 100 characters), e.g.:
`sudoku,puzzle,brain game,logic,numbers,offline sudoku`

**Google Play** — no separate keyword field; use these terms naturally in Short description and Full description.
